Description of Pharma Project

The point of this project was to test and showcase my analytical skills. Using Python libraries (pandas, matplotlib) to analyze technical data in order to create graphs that are easy to read and interpret.
Here you will find me combining test data(csv format) on different cancer treatment methods used on mice. I also arranged and calculated statistical data from these datasets using pandas, and visualized the results using matplotlib.

Findings

1. Capomulin was the most effective treatment shown, reducing average tumor size by ~19% over a 45 day treatment period.
